# Expense Tracker Dashboard

A full-stack expense tracking dashboard built with React, Node.js, Express, and MySQL.

This project allows users to manage expenses, track spending analytics, and view financial insights in a modern SaaS-style dashboard.

## Features
- User authentication (login & register)
- Add, edit, delete expenses
- Dashboard summary cards
- Expense analytics by category and month
- Modern responsive UI

## Tech Stack
Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MySQL

## Installation

Clone the repository

git clone https://github.com/LuckypGen5/expense-tracker-dashboard.git

Install backend

cd backend
npm install
npm run dev

Install frontend

cd frontend
npm install
npm run dev
